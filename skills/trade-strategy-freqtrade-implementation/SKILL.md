# Trade Strategy Freqtrade Implementation Skill

Use this skill when the task starts from coding or revising a Freqtrade strategy.

## Goal

Translate the approved prototype into reproducible Freqtrade code.

## Required Work

- implement timeframe and informative pair handling
- implement indicators and signal mapping
- implement stop-loss, take-profit, and position controls
- keep logic auditable and aligned with the written prototype

## Output

- Freqtrade strategy implementation
- config or artifact dependency notes
- implementation assumptions
